## 📖 Сюжет

> *Ты очнулся без памяти у врат Валенхольма — последнего оплота людей в мире, пережившем Падение. Кто ты? Откуда пришёл? И почему Тьма знает твоё имя?*

> *Найди ответы. Вступи в Гильдию Искателей. Раскрой тайну Завесы. Но будь осторожен — не все, кто улыбается, желают тебе добра.*

Мир изменился после катастрофы, известной как **Падение**. Древние руины хранят забытые знания, а в Туманном лесу бродят теневые волки. Только ты можешь восстановить равновесие — если найдёшь **Осколок Рассвета** и трёх **Хранителей Памяти**.

## 🚀 Установка и запуск

### Требования

- **Python** 3.10+
- **Pygame** 2.5+

### Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/sinnerdevdevdev/echoes-of-the-fallen.git
cd echoes-of-the-fallen

# Установите зависимости
pip install pygame
```

### Запуск игры

```bash
python main.py
```

### Тестовые скрипты

```bash
# Свободная прогулка по миру (с входом в здания)
python test_world.py

# Тест диалоговой системы (5 диалогов)
python test_dialogue.py

# Тест системы уведомлений (клавиши 1-0)
python test_notifications.py

# Галерея всех персонажей и NPC
python test_characters.py
```

## 🎮 Управление

| Клавиша | Действие |
|:-------:|----------|
| `W A S D` | Передвижение |
| `E` | Взаимодействие (NPC / здание) |
| `I` | Инвентарь / характеристики |
| `ESC` | Меню / выход из здания |
| `SHIFT + E` | Выход из здания (в игре) |
| `F1` | Отладочная информация |
| `SPACE` | Продолжить диалог |
| `ПКМ` | Использовать / надеть предмет |
